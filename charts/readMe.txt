To link changes from this repo to the localhost/ 

1. Delete the 'charts' in the HTDOCS directory.
2. Copy the file path of this repo [Ex. "C:\Users\Ellen\GITREPO\B.I-DASHBOARD-\charts"]
3. Copy the file path of htdocs [Ex. "C:\xampp\htdocs\charts"]
4. Open cmd as ADMIN as paste this command: mklink /D "C:\xampp\htdocs\charts" "C:\Users\Ellen\GITREPO\B.I-DASHBOARD-\charts"

Changes from this repo will now be reflected in the localhost/htdocs/charts directory.
